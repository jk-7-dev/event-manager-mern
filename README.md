# Event Manager - MERN Stack

A full-stack Event Booking & Ticket Management application built with the MERN stack (MongoDB, Express, React, Node.js). It features role-based access (Admin/User), secure authentication, QR code ticket generation, and digital verification.

## üöÄ Features

- **Public Event Listing:** Browse upcoming events with a modern, responsive UI.
- **User Authentication:** Secure Login/Register using JWT (JSON Web Tokens).
- **Admin Dashboard:** Create, manage, and delete events (Role-protected).
- **Ticket Booking:** Real-time inventory management prevents overbooking.
- **Digital Tickets:** Generates unique QR codes for every booking.
- **Verification System:** Public endpoint to verify ticket authenticity via QR scan.
- **Responsive Design:** Optimized for Mobile and Desktop with a custom matte UI.
